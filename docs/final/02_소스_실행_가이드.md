# 소스 실행 가이드

## 1. 파일 역할
- `setup_source_env.bat`: Python 3.12 기반 source 가상환경 준비
- `tray_start.bat`: source 실행 진입점
- `tray_start_debug.bat`: 콘솔창을 띄운 상태에서 source 실행 오류 확인

## 2. 현재 확인된 상태
- `setup_source_env.bat`는 현재 정상 완료됨
- `tray_start.bat --check-only` 결과는 아래와 같음
  - `NEW\source\venv_webview\Scripts\pythonw.exe`
- 즉 지금은 source 실행이 release fallback이 아니라 실제 source launcher 기준으로 잡힘

## 3. source 실행 기준
1. `setup_source_env.bat` 실행
2. 필요 시 `tray_start_debug.bat`로 콘솔 확인
3. 평소에는 `tray_start.bat` 사용

## 4. 주의 사항
- source 실행은 Python 3.12 기준으로 유지한다.
- `venv_webview`를 임의 복사하지 않는다.
- source 수정 후 바로 확인할 때는 `tray_start_debug.bat`를 먼저 쓰는 편이 안전하다.
